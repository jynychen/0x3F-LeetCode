## 三、每週節奏建議

- **平日（一～五）**：每天 1–2 題，45–60 分鐘限時。卡關先自己想 15–20 分鐘，看完題解一定要自己重新手打一次，不要複製貼上。
- **週末**：
  - 一場 Weekly/Biweekly Contest，練速度與臨場感。
  - 間隔複習：從「2 週前」「1 個月前」寫過的題目挑 2–3 題重做，對抗遺忘曲線。
- **每題完成後標記三種狀態**：秒殺 / 卡關看解 / 不會。後兩種一律收進「重刷清單」，M10 全面複習月要清空它。

## 四、面試準備 Checklist（M10–M12 重點）

- [ ] 刷 LeetCode Google／大廠公司標籤題（需 Premium）
- [ ] Mock interview：Pramp / interviewing.io，或找同儕互相面
- [ ] System Design（若應徵中高階職位）：*System Design Interview* Vol.1/2 或 Grokking 課程
- [ ] Behavioral：整理 5–8 個 STAR 故事（衝突處理、領導經驗、失敗經驗、跨團隊合作）
- [ ] 履歷量化：每項成果盡量帶數字，並找人 review
- [ ] 內推：LinkedIn／校友／Blind，儘早開始——大廠面試 pipeline 常常拖 1–2 個月
