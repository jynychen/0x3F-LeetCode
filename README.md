## 建議刷題順序

對於初學者，建議按照以下順序循序漸進：

```
二分算法 → 滑動窗口與雙指針 → 位運算 → 單調棧
→ 鏈表、二叉樹與回溯 → 數據結構 → 圖論算法
→ 動態規劃 → 貪心與思維 → 網格圖 → 數學算法 → 字符串
```
