## 方法論摘要

### 核心理念

- **不要死記硬背**，要理解每一類題型的本質思路與適用條件。
- **分專題系統練習**，每個專題集中突破，再進行綜合訓練，比隨機刷題效率高數倍。
- **由易到難**，先把基礎題型做熟，再挑戰困難題，避免在不熟悉的題型上消耗過多時間。
- **複習與回顧**，刷過的題目一段時間後要再做一遍，強化記憶與理解。

### 刷題步驟

1. **選定專題**：按照下方題單，一次專注一個主題。
2. **先思考，再看題解**：獨立思考至少 15–30 分鐘，實在沒有頭緒再看提示。
3. **讀懂題解**：不只看答案，要理解「為什麼這樣做」、「這個方法的適用場景」。
4. **手寫或默寫代碼**：理解後自己重新實作，不要直接複製。
5. **總結模板與套路**：把同一類型題目的解題框架整理成筆記。
6. **定期複習**：利用間隔重複（Spaced Repetition）避免遺忘。

### 常見誤區

- 只刷題數、追求 AC，卻不理解解法。
- 遇到不會的直接看答案，缺少思考過程。
- 跳題亂刷，沒有系統性。
- 做完就忘，不做複習。
